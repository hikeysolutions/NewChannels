## Inheritance (when built)
Will run on Channel A's shared infrastructure only (flyt- agent set, provider registries, QA gate,
assembly pipeline, Telegram approval flow, Section 03c). Will inherit NONE of Channel A's creative
tuning. Persona, prompt content, providers, and visual identity will all be its own.

## Format (concept)
30 to 60 second shorts featuring wholly original character(s), no real people, no existing IP,
dropped into high-stakes, absurd, premium-aesthetic scenarios: VIP events, reality-competition
chaos, high-roller nightlife. Cinematic/luxury visual identity, high-contrast lighting.

## Why This Channel Is Sequenced Last (the operator's own reasoning)
Deliberately last of the five. Unlike the others, its build order runs in reverse: character design
first, then visual style, then everything else. For the other channels the format and pipeline can
be built before the creative specifics are fully locked. Here they cannot, because the whole channel
hangs on a specific original character existing first. There is nothing to build a pipeline around
until that character is designed. Do not start pipeline work ahead of character design.

## The Real Flagged Risk (parasocial attachment)
The hardest unsolved problem across all five channels: building genuine audience/parasocial
attachment to a wholly new character with zero prior recognition. Existing case studies (Lil
Miquela, Neuro-sama) prove it is achievable, so this is not a blocker, but it proves it takes
deliberate technique, specifically consistency-as-authenticity (the character stays visually and
behaviorally identical across appearances until the audience treats that consistency as a real
identity), NOT backstory dumps. A written backstory does not build attachment. Consistency does.
This is the design problem to solve first, before any format or pipeline work.

## Compliance (inherited)
Universal hard rules apply and are already followed: no real-person likeness, no depiction of
minors. Wholly original character(s) only, no existing IP. No new independent policy gate.
